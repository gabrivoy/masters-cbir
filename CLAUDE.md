## Project

CBIR (Content-Based Image Retrieval) - early-stage Python project. Currently a skeleton with a single `main.py` entry point.

## Development Environment

- Python 3.13, managed via `.python-version`
- Uses `uv` for project/dependency management (`pyproject.toml`, `.venv`)
- Run: `uv run main.py`
- Add dependencies: `uv add <package>`
